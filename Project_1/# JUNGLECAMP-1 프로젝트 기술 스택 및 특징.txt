# JUNGLECAMP-1 프로젝트 기술 스택 및 특징

## 프론트엔드

- **HTML**  
  기본 UI 구조 및 입력 폼, 결과 출력 영역 작성

- **CSS**  
  박스, 버튼, 로딩창, 애니메이션 등 스타일링  
  커스텀 폰트(`HeadlinerNo45`) 및 PUBG 관련 이미지 사용

- **JavaScript / jQuery**  
  - UI 이벤트 처리(검색, 클릭, 입력 등)
  - AJAX로 PUBG 공식 API 호출 (실시간 데이터 연동)
  - 결과 애니메이션(총알, 박스 크기 변화, 폭발 등)
  - 에러 처리 및 사용자 안내
  - jQuery 3.5.1 사용

- **JSON 데이터 활용**  
  - PUBG API에서 받은 응답(JSON)을 파싱하여  
    플레이어 정보 및 전적을 동적으로 화면에 출력  
  - 정적 코딩이 아닌, 실시간 API 데이터 기반 동적 UI 구현

---

## 백엔드

- **PUBG 공식 API**  
  - RESTful 방식, Bearer 토큰 인증  
  - 플레이어 정보 및 시즌 전적 데이터 조회  
  - 프론트엔드에서 직접 AJAX로 호출

- **자체 서버 없음**  
  - 별도의 Flask/Django 등 서버 없이  
    프론트엔드에서 API 직접 사용

---

## 기타 특징

- **애니메이션 효과**  
  - CSS/JS로 박스 크기 변화, 파티클, 낙하산, 폭발 등 구현

- **반응형 디자인**  
  - 모바일 대응 미디어 쿼리 적용

- **에러 및 예외 처리**  
  - API 오류, 404 등 상황별 안내 메시지 및 UI 복구

- **실시간/동적 코딩**  
  - JSON 기반 실시간 데이터로 결과를 동적으로 렌더링  
  - 정적 데이터가 아닌, API 응답에 따라 UI가 변화
